# Vehicle Viewer - Three.js

Interactive 3D vehicle viewer created with Three.js and Vite, featuring camera controls, LERP view buttons, HDRI environment, and shadows.

-

## 📦 Requirements

- Node.js >= 20
- npm >= 10
- Modern browser (Chrome, Edge, Firefox, Safari)
- Git (for deployment with `gh-pages`)

---

## 🔧 Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/zurol/vehicle-viewer.git
cd vehicle-viewer
npm install
